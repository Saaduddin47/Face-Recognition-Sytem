# 👨‍🏫 Face Recognition Attendance System

This is a **Flask-based web application** that uses **face recognition** to mark attendance. It captures faces using a webcam, trains a **K-Nearest Neighbors (KNN)** model, and recognizes registered users to log their attendance automatically.

---

## 🚀 Features

- 📷 Detects faces using OpenCV’s Haar Cascade.
- 🧠 Trains a face recognition model using KNN.
- 🗓️ Maintains daily attendance logs in CSV format.
- 👥 Add/Delete users with live webcam capture.
- 🧾 Real-time attendance marking with timestamp.
- 🌐 Web interface using Flask for ease of use.

---

## 🧰 Tech Stack

- **Python**
- **OpenCV**
- **Flask**
- **NumPy, Pandas**
- **Scikit-learn (KNN)**
- **HTML/CSS (Jinja templates)**
